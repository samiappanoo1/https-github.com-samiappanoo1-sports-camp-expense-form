# Sports Camp Expense Form

A modern, interactive **multi-step expense management form** for sports coaching camps. Built with **HTML, CSS, and JavaScript** (no dependencies required).

## 🏅 Features

### ✅ Multi-Step Form (6 Steps)
1. **Details** - Collect institution info, coach, manager, players, venue, and dates
2. **Camp Expenses** - Add/remove coaching camp expense rows with dynamic calculations
3. **Travel** - Add/remove travel routes (Flight, Bus, Train, Auto) with amounts
4. **Daily Allowance** - Auto-calculates: (Players + 2 Staff) × Days × Rate
5. **Other Expenses** - Uniform, registration, officiating, equipment, miscellaneous
6. **Review & Submit** - Full summary with grand total verification

### 🎨 User Experience
- **Progress Bar** with clickable step dots for easy navigation
- **Real-time Calculations** for all expense categories
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Form Validation** - Ensures required fields are filled
- **Beautiful UI** - Gradient background, smooth animations, modern styling

### 📊 Automatic Calculations
- **Camp Subtotal** - Auto-sums all camp expense rows
- **Travel Subtotal** - Auto-sums all travel expense rows
- **Daily Allowance** - Auto-calculates based on: `(Players + 2) × Days × Rate`
- **Grand Total** - Combines all expense categories
- **Reference Number** - Auto-generated on submission
